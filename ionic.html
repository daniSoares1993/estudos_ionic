Ionic

  • O ionic framework é um kit de desenvolvimento de software (SDK),"SOFTWARE DEVELOPMENT KIT" utilizado para desenvolver aplicativos moveis hibridos,com javascript,html,css;

App Hibrido ou Nativo qual a diferença?

Apps hibridos

  • Um app hibrido ele basicamente roda dentro de um celular,porem ele precisa de mais um componente;
  • O ionic utiliza uma WebView que lembra um pouco um browser e dentro dessas WebView é onde o seu app ira rodar;
  • A grande diferença de uma webview é que ela basicamente renderiza a interface do seu app,assim como o browser faz quando criamos um site,porem a grande diferença é que, a webview tambem acessa os recursos nativos do celular oque um browser não consegue fazer,podendo então acessarmos fotos,camera etc...
  • Basicamente,desenvolvemos os codigos html,css,javascript,esses codigos são rodados dentro dessa webview e essa webview que roda dentro do celular;
  • Porque utilizar o hibrido?
    É uma maneira produtiva de voce construir os seus apps;

  • Plataformas

    Android e iOS

Apps nativos

  • Quando criamos um app nativo,ele depois de criado,é rodado, diretamente dentro do celular,ou seja,ele nao possui essa webview;
  • Isso acaba trazendo algumas vantagens de acesso aos recursos do celular,pois como nao temos um recurso intermediario,essa execução é muito mais fluida, ela é muito mais rapida,mais performatica;

Utilizar nativo ou hibrido ?

  • Nativo

  • Depende do projeto,caso seu projeto utilize muitos recursos nativos,por exemplo,geolocalização,câmera etc...talvez seria interessante desenvolver de forma nativa; ou seja,como falamos a forma nativa é muito mais performatica;

  • Hibrido

  • Caso você for desenvolver algum app que consuma muitos recursos de site,ai sim seja mais vantajoso utilizar o ionic

Vantagens de utilizar o Ionic

  • Utilizamos html,css,javascript para criar o app;
  • Quando desenvolvemos apps para android nos utilizamos a linguagem Java e utilizamos android studio, ou seja, os devs precisariam aprender a linguagem java e utilizar o android studio para desenvolver o app; O mesmo acontece para o iOS a pessoa precisaria aprender swift por exemplo e utilizar uma IDE que no caso seria o xcode;
  • Produtividade(temos o mesmo codigo para ambas as plataformas)

Comandos iniciais

  • ionic start myApp tabs --type para definir o framework angular ou react
    myApp = blank,tabs,sidemenu
  • blank => basicamente é um projeto vazio
  • tabs => projeto com abas para você navegar em seu projeto
  • sidemenu => cria um projeto com um menu lateral
  • Apos escolher o framework recebemos uma pergunta: se você quer integrar seu novo aplicativo com o "CAPACITOR",(ferramenta que permite a criação do app com ionic) e gere uma versao nativa para executar com android studio ou ios com xcode;

    • Ionic generate

  • Ionic generate => pergunta oque voce gostaria de gerar e nos tras algumas opcções; podemos gerar diretamente como por exemplo: ionic generate oque desejamos gerar e o nome do arquivo que iremos gerar;

  • iniciando projeto

  • ao executar o comando ionic serve iremos iniciar o nosso projeto; PS: lembrando que devemos estar com a pasta aberta no terminal

Estrutura do ionic

  • ion header = basicamente cria a area superior(cabeçalho); por padrao ja vem o header com position sticky top
  • ion tooblar = cria uma toolbar; A Barra de ferramentas é um componente utilizado pelos softwares com interface gráfica com a finalidade de permitir uma ação rápida por parte do usuário, facilitando o acesso a funções do programa.
  • ion title = seria nosso logo
  • Ou seja da sombra pra cima ion header da sombra pra baixo ion content
  • ion content = basicamente define toda a area de conteudo;
  • ion content = onde iremos desenvolver a estrutura de codigos que ja estamos mais familiarizados
  • ion footer = nosso rodapé
